Git - Komendy

ABY UPDATOWAĆ NA DYSKU WPISUJĘ:

  git commit -m "nazwa zmiany" -a

FLAGA   -m   oznacza "message"
FLAGA   -a   oznacza, że   commit   ma objąć wszystkie zmiany.
NASTĘPNIE WYSYŁAM DO NETA, DO REPOZYTORIUM NA GITHUBIE, WPISUJĘ:

  git push

podaję dane logowania do GitHuba

Takim sposobem się updatuje repozytorium.


ŚCIĄGANIE Z GITHUBA:

  git pull
